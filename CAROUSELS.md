# PULSE — Логика трёх каруселей

> Официальная спецификация. Какая карусель что показывает.
> Последнее обновление: 2026-05-27

---

## Общий принцип

| # | Название | Что показывает | Для кого |
|---|----------|---------------|----------|
| 1 | **"Это вы ещё не видели"** | Непрочитанные новости по тегам пользователя | Только залогиненным |
| 2 | **"Вся лента"** | Прочитанные новости по тегам (хронология) | Только залогиненным |
| 3 | **"Общая лента"** | Все новости из базы (без фильтра тегов) | Всем |

**Логика:**
- 1-я карусель = **новое** (ещё не видел)
- 2-я карусель = **история** (уже видел, можно вернуться)
- 3-я карусель = **полная лента** (всё что есть в базе)

---

## Карусель 1: "Это вы ещё не видели"

### Что показывает
Новости, которые:
1. Совпадают с тегами пользователя (`matched_tags && user_tags`)
2. Пользователь **ещё не видел** (нет в `user_news_reads`)
3. Опубликованы за последние 90 дней

### Сортировка
Новые сверху (`published_at DESC`)

### API
```
GET /api/news?limit=50
```

### Код компонента
`UnreadNewsCarousel.tsx`

### Auto-mark-read
Когда карточка появляется в viewport (IntersectionObserver) → отправляется `POST /api/news/:id/read` → новость исчезает из этой карусели и появляется во 2-й.

---

## Карусель 2: "Вся лента"

### Что показывает
Новости, которые:
1. Совпадают с тегами пользователя (`matched_tags && user_tags`)
2. Пользователь **уже видел** (есть в `user_news_reads`)
3. Опубликованы за последние 90 дней

### Сортировка
Хронологический порядок (`published_at ASC`) — старые слева, новые справа

### API
```
GET /api/news?history=true&limit=50
```

### Код компонента
`AllNewsCarousel.tsx`

### Логика
Это **история просмотров**. Пользователь может вернуться к уже прочитанным новостям. Первая карусель показывает новое — вторая показывает что уже было просмотрено.

---

## Карусель 3: "Общая лента"

### Что показывает
Все новости из базы данных:
1. **Без фильтра по тегам** — любые новости
2. Опубликованы за последние 90 дней

### Сортировка
Новые сверху (`published_at DESC`)

### API
```
GET /api/news?global=true&limit=50
```

### Код компонента
`GlobalNewsCarousel.tsx`

### Логика
Даже если у пользователя нет тегов — он видит общую ленту всех новостей. Это даёт представление о том, какие новости собирает система.

---

## Визуальная схема

```
┌──────────────────────────────────────────────┐
│  КАРУСЕЛЬ 1: "Это вы ещё не видели"         │
│  ┌─────┐ ┌─────┐ ┌─────┐                   │
│  │ NEW │ │ NEW │ │ NEW │  ← непрочитанные  │
│  └─────┘ └─────┘ └─────┘    по тегам       │
│                                              │
│  КАРУСЕЛЬ 2: "Вся лента"                    │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐          │
│  │ old │ │     │ │     │ │ new │  ← прочитанные │
│  └─────┘ └─────┘ └─────┘ └─────┘    по тегам, хронология │
│                                              │
│  КАРУСЕЛЬ 3: "Общая лента"                  │
│  ┌─────┐ ┌─────┐ ┌─────┐                   │
│  │ ALL │ │ ALL │ │ ALL │  ← все новости    │
│  └─────┘ └─────┘ └─────┘    без фильтра    │
└──────────────────────────────────────────────┘
```

## Flow прочтения

```
Пользователь скроллит карусель 1
    │
    ▼
Новость появляется в viewport
    │
    ▼
POST /api/news/:id/read
    │
    ▼
┌─────────────────────┐
│ Новость исчезает    │
│ из карусели 1       │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ Новость появляется  │
│ в карусели 2        │
│ (хронология)        │
└─────────────────────┘
```

## API Summary

| Карусель | Endpoint | Фильтр | Сортировка |
|----------|----------|--------|------------|
| 1. "Не видели" | `GET /api/news` | теги + непрочитанные | DESC |
| 2. "Вся лента" | `GET /api/news?history=true` | теги + прочитанные | ASC |
| 3. "Общая" | `GET /api/news?global=true` | без фильтра тегов | DESC |

## Компоненты

| Карусель | Файл |
|----------|------|
| 1 | `UnreadNewsCarousel.tsx` |
| 2 | `AllNewsCarousel.tsx` |
| 3 | `GlobalNewsCarousel.tsx` |

## Условия отображения

```tsx
// Home.tsx
{isLoggedIn && selectedTags.length > 0 && <UnreadNewsCarousel />}  // 1
{isLoggedIn && selectedTags.length > 0 && <AllNewsCarousel />}      // 2
<GlobalNewsCarousel />                                               // 3 (всегда)
```
