## TC-1: Создание аккаунта (Регистрация)

**Предусловие:** Пользователь не авторизован, открыта главная страница.

**Шаги:**
1. Кликнуть **"Начать"** в навбаре
2. Убедиться, что попап открылся на вкладке **"Регистрация"**
3. Ввести **Логин**: `newuser2025`
4. Ввести **Email**: `newuser2025@test.com` (уникальный)
5. Ввести **Пароль**: `StrongPass1!`
6. Убедиться, что PasswordStrength показывает **4/4 "Отличный"**
7. Ввести **Подтвердите пароль**: `StrongPass1!`
8. Поставить галочку **"Я согласен с Условиями"**
9. Кликнуть **"Создать аккаунт"**

**Ожидаемый результат:**
- ✅ Попап переключается на **Success-экран**
- ✅ Зелёная иконка CheckCircle (bounce-анимация)
- ✅ Текст: "Аккаунт создан!"
- ✅ Кнопка **"Начать"** видна
- ✅ Клик "Начать" закрывает попап
- ✅ Пользователь авторизован (виден username в навбаре)

**Очистка после теста:**
```bash
# Найти и удалить тестового пользователя из БД
```

## TC-3: Добавление и удаление тега

**Предусловие:** Пользователь залогинен (TC-2 пройден).

**Шаги:**

2. Кликнуть в поле поиска
3. Ввести: `Сбер`
4. Выбрать **"Сбербанк"** из dropdown
5. Наблюдать за анимацией добавления (tag-loading glow)

**Ожидаемый результат:**
- ✅ Тег "Сбербанк" появляется в ряду тегов
- ✅ Счётчик тегов: **1/3** (голубой)
- ✅ Тег анимирован (pulse glow 1.5 сек)
- ✅ Новости по тегу загружаются

### 3B: Добавление второго тега

2. Выбрать **"Газпром"**

**Ожидаемый результат:**
- ✅ Тег "Газпром" добавлен
- ✅ Счётчик: **2/3**

## TC-6: Дизайн попапов (визуальная проверка)

| Элемент | Критерий |
|---------|----------|
| **Backdrop** | Размытие (`blur-sm`) + тёмный слой |
| **Логотип** | Размер 48px (`h-12`), не менее |
| **Табы** | Pill-форма, плавное переключение |
| **Input иконки** | Mail, Lock, User — слева, цвет `#6B7280` |
| **Глазок пароля** | Eye/EyeOff — справа, toggle показа |
| **PasswordStrength** | 4 сегмента, цвета: красный→жёлтый→зелёный→голубой |
| **Ошибки** | Красный текст, анимация появления |
| **Success-экран** | CheckCircle bounce, staggered анимация элементов |
